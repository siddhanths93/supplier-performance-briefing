# Supplier Performance & Concentration Briefing Tool

A Python analytics application that converts synthetic supplier spend and
performance data into a ranked management-attention agenda.

## Project goals

The tool will eventually:

- analyze supplier spend and performance,
- identify concentration and fragmentation,
- rank suppliers requiring management attention,
- generate evidence-backed findings,
- present results through a local Streamlit application,
- export executive-ready analytics.

## Technology stack

- Python
- Pandas
- NumPy
- Plotly
- Streamlit
- OpenPyXL
- Pytest

## Project status

Phase 1: Initial project setup.
